# Q112 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on veto power and how it differs from value capture. A party holds a veto if its withdrawal
collapses the coordination. Q104 found every edge of the minimal triad load-bearing, so every core party
may veto; Q111 found value concentrated at the mediator. This asks whether both hold, so that destruction is
democratic while value is concentrated. Written and committed before the run; the instrument was validated.

## H1 — Every core party holds a veto

- **Claim:** Each of the worker, mediator, and recipient collapses the triad by defecting (going constant).
- **H0:** Some core party's defection leaves the form triadic.
- **Predicted outcome:** all veto. H0 refuted. Every bound party can unilaterally break the coordination.

## H2 — A non-core party holds no veto

- **Claim:** A read-only spectator's defection leaves the major complex {E, M, R} intact.
- **H0:** The spectator can collapse the core.
- **Predicted outcome:** no veto. H0 refuted. Only bound parties hold a veto.

## H3 — Veto is universal but value is not

- **Claim:** All three core parties hold a veto, yet their Shapley values are unequal (Q111).
- **H0:** Veto power and value capture coincide.
- **Predicted outcome:** universal veto, unequal value. H0 refuted. Destruction is democratic; value is
  concentrated at the mediator.

## H4 — The coordination is not a unanimity game

- **Claim:** The sub-coalition values are asymmetric — the recipient pair without the mediator is worthless
  while a mediator pair is not — which is why the mediator captures disproportionate value despite the
  universal veto.
- **H0:** The sub-coalition values are symmetric (a unanimity game).
- **Predicted outcome:** asymmetric. H0 refuted. A unanimity game would split value equally; the mediator's
  productivity in sub-coalitions is what concentrates the value.

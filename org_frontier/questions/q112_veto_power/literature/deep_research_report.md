# Q112 literature — veto power and the value it does not track

## Veto players

A veto player in a cooperative game is one whose participation every winning coalition needs. Q104 found
every edge of the minimal triad load-bearing and Q107 that repair is lever-specific, both pointing to a
coordination held by conditions each of which is necessary. The game-theoretic reading is that every core
party is a veto player: withdraw any one and the coordination collapses.

## Veto against value

Q111 found value concentrated at the mediator, which captures two-thirds. If every core party holds a veto
but only the mediator captures a large share, then veto power and value capture are distinct: the power to
destroy is held equally, while the value is held unequally. The reason would be in the sub-coalition values
— the two outer parties produce nothing together, while either produces the full value with the mediator —
so the coordination is not a unanimity game, where value splits equally among veto players, but a game where
the mediator's productivity in partial coalitions concentrates the value. Destruction is democratic; rent is
not.

## Method context

Veto is read by defecting each party to a constant and recomputing the verdict; value is the Q111 Shapley
value; the sub-coalition values are the Q111 value function on the pairs.

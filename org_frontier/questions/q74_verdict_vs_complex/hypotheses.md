# Q74 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on the divergence between the whole-system verdict and the maximal complex, over four
forms from the program: a disclosed triad (Q63), a delegated exchange (Q69), a monitored triad (Q68), and
an agent chain (Q65). Written and committed before any test runs.

## H1 — Spectator and frozen forms are whole-system dyadic but triadic on the core

- **Claim:** The disclosed (D'=M readout), delegated (frozen E, R), and monitoring (T'=M readout) forms
  are whole-system dyadic (Φ_MIP = 0), while their maximal complex is triadic (Φ > 0).
- **H0:** Some such form is whole-system triadic, or has a dyadic maximal complex.
- **Predicted outcome:** all three whole-system dyadic, maximal complex triadic. H0 refuted.

## H2 — In those forms, every excluded element is non-bidirectionally coupled

- **Claim:** In the three forms above, each element outside the maximal complex is either emit-only (feeds
  the core but reads no core element) or read-only (reads the core but is read by none) — not
  bidirectionally coupled.
- **H0:** Some excluded element is bidirectionally coupled to the core.
- **Predicted outcome:** every excluded element non-bidirectional. H0 refuted.

## H3 — The chain is whole-system triadic with a bidirectionally-coupled excluded element

- **Claim:** The agent chain is whole-system triadic, its maximal complex is a proper subset, and at least
  one excluded element is bidirectionally coupled to the core (coupling necessary, not sufficient).
- **H0:** The chain is whole-system dyadic, or its excluded elements are all non-bidirectional.
- **Predicted outcome:** whole-system triadic, maximal complex a proper subset, an excluded element
  bidirectional. H0 refuted.

## H4 — Excluded-element coupling predicts the divergence type

- **Claim:** Across the four forms, an excluded element is bidirectionally coupled if and only if the form
  is whole-system triadic (divergence type B); whole-system dyadic with a triadic core (type A) occurs
  exactly when every excluded element is non-bidirectional.
- **H0:** The coupling of excluded elements does not predict the divergence type.
- **Predicted outcome:** the biconditional holds across the four forms. H0 refuted.

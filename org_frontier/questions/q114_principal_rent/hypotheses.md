# Q114 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on a principal's value capture. Finding 8 found a principal joins the core only when
bidirectionally coupled. This asks whether joining is worth value, and whether the principal's value comes
at the parties' expense (rent) or adds to the total (creation) — the discriminator the agenda set. Written
and committed before the run; the instrument was validated.

## H1 — A bidirectional principal captures positive value

- **Claim:** A principal bidirectionally coupled to the mediator has positive Shapley value.
- **H0:** Its value is zero or negative.
- **Predicted outcome:** positive. H0 refuted. Joining the core is worth something.

## H2 — A monitor-only principal captures non-positive value

- **Claim:** A principal that reads the mediator but is not read by it has Shapley value at most zero.
- **H0:** It captures positive value.
- **Predicted outcome:** non-positive. H0 refuted. Ownership without productive coupling is not value, and a
  read-only principal that factors the whole system holds negative value.

## H3 — The principal's value is added, not extracted

- **Claim:** When the bidirectional principal joins, no existing party's Shapley value falls below its value
  in the base triad.
- **H0:** Some party's value falls — the principal extracts rent from the parties.
- **Predicted outcome:** no party falls. H0 refuted. The principal creates value rather than extracting it.
  This is the study's genuinely uncertain claim — a principal could equally capture value at the parties'
  expense.

## H4 — The principal grows the total value

- **Claim:** The coordination's Φ with the bidirectional principal exceeds its Φ without (2.0).
- **H0:** The total does not grow.
- **Predicted outcome:** it grows. H0 refuted. A productive principal enlarges the coordination's value.

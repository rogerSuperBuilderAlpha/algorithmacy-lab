# Q66 — Stage 1 review: where the core sits in an agent chain

## The question

Q65 found that an agent chain E → A1 → … → Ad → R stays triadic at every depth, but its irreducible core
is not the whole chain: at d = 2 the maximal complex is the recipient-facing pair {A2, R}, with the
sender and upstream agent feeding it without joining. This study asks where the core boundary sits in
general, and what moves it. Does the core stay a two-element recipient-facing pair as the chain
lengthens, and does closing the chain into a loop pull the whole structure into the core?

## What the lab already knows that bears on this

- **Q65's localization (probes 225–228).** The d = 2 chain's maximal complex is {A2, R} at Φ = 2.0, not
  the whole chain. This study generalizes across depth and tests interventions on the boundary.
- **Core membership needs bidirectional coupling (STRUCTURAL_FINDINGS, probe loop).** A party is in the
  irreducible core only if it both feeds and is fed by the coordination; emit-only sources and read-only
  sinks stay out. The chain's interior agents are bidirectionally coupled, yet Q65 found them outside the
  maximal complex, so coupling is necessary but not sufficient: pivotality decides among coupled elements.
- **A feedback cycle, not a directed path, carries irreducibility (q43; Finding on closed loops).** A
  directed cycle in the wiring is a different object from a closed loop in the causal structure. Closing
  the chain into a ring should bind the whole structure where an open chain does not.

## The gap

Q65 established the d = 2 localization but did not test whether it holds across depth, whether the core
stays size two, or whether closing the chain into a loop enlarges the core. No prior probe maps the core
boundary of the agent chain, so the question is open.

# Q67 — Stage 1 review: the reciprocity gradient

## The question

Q63 found that one-shot outreach with a frozen recipient is dyadic (liveness is required). Q66 found that
a one-way agent chain has a two-element end-pair core, while closing it into a loop binds the whole
structure at Φ = 4.0. Between a pure feed-forward relay and a fully closed loop lies a gradient of
reciprocity. This study asks how the verdict and the irreducible core change as reciprocity is added to a
four-element agent exchange (E, A1, A2, R): does whole-system triadicity arrive gradually, and does the
core grow smoothly or jump at loop closure?

## What the lab already knows that bears on this

- **Liveness is necessary (Q63 H5, probe 219).** A read-but-frozen recipient leaves outreach dyadic. A
  pure feed-forward relay with no feedback should be dyadic.
- **Open chain core is an end pair; closed ring binds all (Q66, probes 229–230).** The one-way chain's
  core is two elements; closing the loop makes the core the full set at Φ = 4.0. This study fills in the
  levels between.
- **A closed causal loop carries irreducibility a directed path does not (q43; closed-loop finding).**
  Reciprocity is what closes the loop, so the transition to a whole-structure core should track loop
  closure.

## The gap

The lab has the endpoints, a relay (dyadic) and a closed ring (full core), but not the gradient between.
Whether a single reciprocal loop suffices for whole-system triadicity, and whether the core grows
gradually or jumps at closure, are uncomputed. No prior probe maps the reciprocity gradient, so the
question is open.

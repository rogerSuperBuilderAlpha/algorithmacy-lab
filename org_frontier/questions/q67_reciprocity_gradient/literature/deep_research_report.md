# Q67 — Stage 2 deep research: reciprocity and closure

This study fills the gradient between Q66's endpoints and draws on the same literatures (shared
`references.bib`). The specific question, how reciprocity changes integration, is internal to the
integrated-information framework.

## Reciprocity and feedback in coordination

Coordination theory treats two-way dependence as the harder case of managing dependencies
[malone1994interdisciplinary]. In communication, a one-way send is a broadcast and a two-way exchange is a
conversation; the economics of spam is about the one-way case, where sending is free and no response is
sought [rao2012economics]. AI-mediated communication spans both: an agent may compose a one-way message or
mediate a back-and-forth [hancock2020aimc; jakesch2019aimc]. The structural question is at which point the
back-and-forth becomes one irreducible coordination.

## Complexes under partial feedback

Integrated information theory identifies the maximal complex as the locally maximal subset
[oizumi2014phenomenology; albantakis2023iit4; mayner2018pyphi]. A system with partial feedback can contain
a small complex while the whole still factors, which this study tests directly: a single reciprocal loop
embedded in a feed-forward relay. The closed-loop versus directed-path distinction the program has found
elsewhere predicts that whole-system irreducibility tracks loop closure.

## The open gap

Q66 gave the endpoints; the gradient between is uncomputed. Whether partial reciprocity yields whole-system
triadicity, and whether the core jumps or grows at closure, is open. In-silico, with the validation gap
[axtell1996aligning]; four hypotheses fixed before computing [chamberlin1965method; platt1964strong;
brodeur2024preregistration].

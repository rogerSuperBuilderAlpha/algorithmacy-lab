# Q113 — Stage 1 review: substitutability and value

## The question

Finding 5 makes substitutability fatal to the verdict. Q111 gave a value measure. What does substitutability
do to value — lower a party's share, or remove the coordination's value entirely? And does an all-required
market distribute value as the mediated triad did, or differently?

## What the lab already knows that bears on this

- **Substitutability collapses the triad (Finding 5, Q85).** The substitutable market is dyadic, so its
  total value is zero.
- **Value concentrates at the mediator of a triad (Q111).** Whether a symmetric all-required market
  concentrates or distributes value is open.
- **The all-required market is triadic with Φ growing in N (Q85).** It has value to distribute; how is the
  question.

## The gap

No study applies the value measure to a market. Whether substitutability erodes a party's value or destroys
the whole coordination's, and whether a required set shares value equally, is uncomputed.

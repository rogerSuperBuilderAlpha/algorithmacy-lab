# Q92 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on a mediator with memory. Every determination in the corpus is memoryless; Q92 gives the
mediator a memory node and asks whether a remembered value of the recipient can substitute for a live read
of it. Finding 3 made liveness — the mediator reading the parties as they currently are — necessary for the
triad. Written and committed before the memory variants are read. The live triad (triadic, Φ = 2) is the
instrument control.

## H1 — A tracking memory preserves the triad

- **Claim:** When the mediator reads a memory that tracks the recipient (M'=E∧Mem, Mem'=R) instead of the
  recipient directly, the form is triadic.
- **H0:** It is dyadic; the one-step delay breaks the binding.
- **Predicted outcome:** triadic — a memory that tracks the recipient substitutes for a live read, so
  liveness is about the recipient being bound through some path, not the current read. This is the study's
  genuinely uncertain claim; the delayed relay could equally break irreducibility, as the relay-gap agent
  did in Q65.

## H2 — A frozen memory collapses the triad

- **Claim:** When the memory does not track the recipient (Mem'=Mem), the form is dyadic.
- **H0:** It is triadic without the memory tracking the recipient.
- **Predicted outcome:** dyadic — a memory substitutes for a read only when it tracks the recipient; a
  stale memory leaves the recipient unbound.

## H3 — Self-memory alone does not make a triad

- **Claim:** When the mediator reads the worker and its own previous state, with the recipient feeding
  nothing (M'=E∧M), the form is dyadic.
- **H0:** Self-reference makes the form triadic without the recipient.
- **Predicted outcome:** dyadic — a mediator's own history does not manufacture a triad without the
  recipient.

## H4 — The tracking memory is in the core

- **Claim:** In the tracking-memory form, the memory node is in the major complex.
- **H0:** The memory node is outside the core.
- **Predicted outcome:** in the core — it carries the recipient's constraint into the mediator.

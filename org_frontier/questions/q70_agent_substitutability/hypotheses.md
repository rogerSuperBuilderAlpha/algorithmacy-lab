# Q70 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on agent substitutability in outreach (worker E, agent(s) M, counterpart C). Written and
committed before any test runs.

## H1 — The single-agent triad is triadic

- **Claim:** Outreach through one agent (E'=M, M'=E∧C, C'=M) is triadic at Φ_MIP = 2.0.
- **H0:** The single-agent form is dyadic.
- **Predicted outcome:** triadic, Φ_MIP = 2.0. H0 refuted.

## H2 — Multi-homing across interchangeable agents collapses the triad

- **Claim:** When the worker and counterpart coordinate through either of two agents (E and C read
  M1∨M2), the form is dyadic.
- **H0:** The substitutable-agent form is triadic.
- **Predicted outcome:** dyadic, Φ_MIP = 0. H0 refuted.

## H3 — Requiring both agents keeps the triad

- **Claim:** When both agents are required (E and C read M1∧M2), the form is triadic.
- **H0:** The required-both form is dyadic.
- **Predicted outcome:** triadic, Φ_MIP > 0. H0 refuted.

## H4 — Both required agents are in the core

- **Claim:** In the required-both form, the major complex contains both agents M1 and M2.
- **H0:** At least one required agent is outside the major complex.
- **Predicted outcome:** core contains M1 and M2. H0 refuted.

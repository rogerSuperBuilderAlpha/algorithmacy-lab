# Q70 — Stage 1 review: agent substitutability

## The question

A worker who multi-homes across interchangeable AI agents, where either agent can carry the outreach,
treats the agent as swappable. This study asks whether that interchangeability collapses the coordination.
Parties: worker E, agent(s) M, counterpart C. When the worker and counterpart coordinate through either
of two agents, is the form still triadic, or does substitutability of the agent collapse it as it does for
other roles?

## What the lab already knows that bears on this

- **Substitutability of any role collapses the triad (Finding 5).** A counterpart bound through a
  disjunction factors; only an all-required conjunction stays triadic. The same holds when the mediator is
  substitutable: a worker multi-homing across two platforms where either suffices factors. This study
  applies that finding to the agent as the substitutable role.
- **When both delegate, the agents are the core (Q69, probe 233).** With agents at the centre of the
  coordination, their substitutability becomes the operative question for whether the coordination holds.
- **The single-agent outreach triad is triadic (Q63).** E-M-C with the agent jointly determining is
  triadic at Φ = 2.0; this is the baseline.

## The gap

The lab established substitutability collapse for counterparts and platforms, but has not built the
interchangeable-agent case in the outreach frame and confirmed it collapses while a required pair stays
triadic. No prior probe answers whether agent multi-homing collapses the coordination, so it is open.

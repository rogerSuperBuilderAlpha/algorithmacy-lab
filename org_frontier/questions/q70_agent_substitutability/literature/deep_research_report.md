# Q70 — Stage 2 deep research: interchangeable agents

This study applies the substitutability finding to AI agents and draws on the same literatures (shared
`references.bib`).

## Multi-homing and interchangeability

Platform research treats multi-homing, using several interchangeable providers, as a force that weakens
any one provider's hold [malone1994interdisciplinary]. In outreach, a worker who routes through whichever
AI agent is cheapest or available treats agents as interchangeable. AI-mediated communication is moving
toward many interchangeable agent providers [hancock2020aimc; jakesch2019aimc], so whether
interchangeability changes the structure of the coordination is the question.

## Substitutability and irreducibility

Costly-signaling theory notes that an interchangeable, low-commitment channel carries little
[spence1973job; zahavi1975mate], and the economics of spam turns on cheap interchangeable sending
[rao2012economics]. The lab's structural counterpart is Finding 5: substitutability of any role collapses
the triad, read by exact integrated information [oizumi2014phenomenology; albantakis2023iit4;
mayner2018pyphi]. This study tests it for the agent role.

## The open gap

The substitutability finding is established for counterparts and platforms; the interchangeable-agent case
in the outreach frame is uncomputed. The gap is open: build single, substitutable, and required-both
agent forms and read the verdict. In-silico, with the validation gap [axtell1996aligning]; four hypotheses
fixed before computing [chamberlin1965method; platt1964strong; brodeur2024preregistration].

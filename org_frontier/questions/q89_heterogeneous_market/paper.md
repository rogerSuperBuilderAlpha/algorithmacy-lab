# Q89 — Which heterogeneous agents enter the core of a market?

## Question

Q85 scaled the agent market to N interchangeable agents and found it collapses all-or-nothing: a
substitutable market is a broadcast at every size, and only an all-required set stays triadic, with every
agent in the core. Real markets are not symmetric. This study makes the agents heterogeneous (a full
agent reading both outer parties, partial agents reading one, a passive agent read by none) and asks which
enter the major complex.

## Method

Worker E, agents M1.., counterpart C. Each agent reads by type: full (E∧C), sender (E), or recipient (C).
The outer parties read forward a chosen active set of agents through AND (all-required) or OR
(substitutable); agents outside the active set are passive. Six configurations are classified by exact Φ
and read on the major complex. The all-required full pair and the substitutable full pair reproduce Q85 and
serve as the control.

## Results

The control holds: two full required agents give a triadic market at Φ = 4.0 with core {E, M1, M2, C}. A
passive agent stays out, leaving the core {E, M1, C} (H1). A substitutable heterogeneous market is dyadic
(H4).

A required partial reader does not enter the core. The sender-only market is triadic but its core is
{M1, C}, excluding the sender-only agent and the worker. The recipient-only market mirrors it with core
{E, M1}. Yet in the three-agent mixed market, where a full, a sender-only, and a recipient-only agent are
all required, the same partial agents are all present, in a triadic market at Φ = 2.0 with the full
five-element core.

| configuration | whole-system | Φ | major complex |
|---|---|---|---|
| full pair, all-required | triadic | 4.00 | {E, M1, M2, C} |
| one passive agent | dyadic | 0.00 | {E, M1, C} |
| full + sender-only | triadic | 1.00 | {M1, C} |
| full + recipient-only | triadic | 1.00 | {E, M1} |
| full + sender + recipient | triadic | 2.00 | {E, M1, M2, M3, C} |
| substitutable | dyadic | 0.00 | {E, M2} |

| | result |
|---|---|
| H1 passive agent excluded | confirmed |
| H2 sender-only required agent enters | refuted |
| H3 mixed market is one stable full core | refuted |
| H4 substitutable market dyadic | confirmed |

## Interpretation

Membership of a heterogeneous agent is a property of the joint determination, not of the agent alone. A
partial reader that, on its own coupling, looks bound (bidirectional and required) can still sit outside
the core when a full partner already covers the determination, in which case its reading is redundant and
the core localizes to the irreducible subset that excludes it. The same partial reader becomes pivotal, and
enters, when the other agents leave a gap only it fills. The two-agent and three-agent markets give
opposite verdicts on the same agent type for this reason.

The result refines Finding 8 without overturning it. Bidirectional coupling stays necessary and
substitutability stays fatal: a passive agent never enters and an interchangeable market always collapses.
What is not a per-agent property is pivotality. Whether a bidirectional, required agent is pivotal depends
on what the rest of the market supplies, so the membership of heterogeneous agents cannot be read off each
agent's local wiring and must be computed on the whole form.

For agent-mediated coordination, the lesson is that adding a capable agent does not by itself add a member
of the core. An agent joins the irreducible coordination only when the determination is irreducible without
it, which depends on the other agents present. A market can hold many bidirectional agents and bind only
the few whose reading no one else covers.

## Limitations

In-silico; Boolean models with exact Φ and exact major complex. The configurations are a handful of curated
forms at N = 2 and 3 agents, chosen to isolate the composition effect, so the
finding is demonstrated rather than mapped exhaustively. The one-passive market is whole-system dyadic with
a triadic major complex; the major complex is the membership reading, governed by the Q74 rule. Agent types
are limited to full, sender-only, and recipient-only reads.

# Q89 literature — heterogeneous agents in a coordination market

## The gap Q85 left

Q85 scaled the agent market to N interchangeable agents and found it collapses all-or-nothing: a
substitutable market is a broadcast at every size, and only an all-required set stays triadic, with every
agent in the core. Its caveat marks heterogeneous agents as untested. Real agent markets are not
symmetric: a routing agent reads both parties, a logging agent reads one, a monitor reads without being
read. Whether a heterogeneous market selects a stable core, or collapses like the interchangeable one, is
open.

## What the law predicts

Finding 8 and Q90 give the membership rule: a party is in the core if and only if it is bidirectionally
coupled, and its membership probability rises with pivotality. Applied to heterogeneous agents, the rule
predicts selection: a passive agent that is read by nothing is not bidirectional and should stay out,
while a required agent that reads even one outer party is bidirectional and should enter. Whether reading
a single party suffices, or whether membership needs reading both, is the sharp question. The market frame
also tests whether substitutability still collapses the form when the agents are unlike each other
(Finding 5).

## Method context

The forms extend Q85's market builder to typed agent reads and an explicit active set, with the verdict
and major complex read by the same instrument used throughout the program.

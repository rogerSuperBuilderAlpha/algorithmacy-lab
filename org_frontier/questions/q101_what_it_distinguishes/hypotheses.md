# Q101 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on what a coordination distinguishes. A distinction specifies a state of its purview, the
configuration the mechanism tells apart. The read-recipient triad's binding distinction (the mediator with
purview the two parties) specifies the joint state (1, 1). This study reads the specified joint state of
the binding distinction across triadic forms and dyadic controls. Written and committed before the
all_required and market forms are read. The read-recipient binding distinction (effect state (1, 1)) is the
instrument control.

## H1 — The binding distinction specifies a multi-party joint state

- **Claim:** The read-recipient triad's binding distinction specifies a state over a purview of two or more
  parties — a specific joint configuration.
- **H0:** Its purview is a single party.
- **Predicted outcome:** a multi-party joint state. H0 refuted. Coordination distinguishes a joint
  configuration of the parties, not a single party's state.

## H2 — The distinguished state is the determination-firing configuration

- **Claim:** The binding distinction's specified effect state is all-ones — the configuration where the
  joint determination fires.
- **H0:** It is some other configuration.
- **Predicted outcome:** all-ones. H0 refuted. The triad distinguishes the state where coordination
  succeeds.

## H3 — Dyadic forms distinguish no joint state

- **Claim:** No dyadic form has a binding distinction specifying a multi-party joint state.
- **H0:** Some dyadic form specifies a joint state.
- **Predicted outcome:** none do. H0 refuted. The joint-state distinction is the semantic signature of
  coordination.

## H4 — Every triad distinguishes the all-present success state

- **Claim:** Across the triadic forms, the binding distinction's specified effect state is all-ones.
- **H0:** Some triad distinguishes a different joint state.
- **Predicted outcome:** all-ones everywhere — coordination uniformly distinguishes the configuration where
  every party is present and the determination fires. Genuinely uncertain for the market and the all-required
  form, which could specify a different joint state.

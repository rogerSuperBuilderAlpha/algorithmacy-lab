# Q84 — Stage 1 review: influence versus membership

## The question

Can an agent positioned outside the irreducible core change the coordination's verdict, or does influence
require membership? This asks whether an adversarial agent can manipulate the dyadic/triadic verdict from
the outside, by reading or feeding the coordination without joining it. Parties: the read-recipient triad
(E, M, R) plus an added agent X in three couplings.

## What the lab already knows that bears on this

- **Non-bidirectional excluded elements stay out, and the core is stable (Q74, Q75, Finding 8).** A
  read-only or emit-only element is a spectator; the core is read on the maximal complex and is robust to
  spectators.
- **The reporting rule (Q74).** A read-only or emit-only addition makes the whole system dyadic while the
  maximal complex stays triadic; the core verdict is the one to read, per the rule.
- **A bidirectional, pivotal element joins the core (Q68, Q74).** Membership requires bidirectional
  coupling plus pivotality.

## The gap

The lab has the membership conditions; it has not framed them as an adversarial question: whether a party
outside the core can change the verdict without joining. No prior probe tests influence-versus-membership
for an added agent, so the question is open.

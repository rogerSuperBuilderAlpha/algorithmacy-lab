# Q69 — Stage 1 review: the two-sided agent exchange

## The question

When both parties delegate to agents, a sender's agent composes for a recipient's agent that triages
before either human acts. The exchange has four elements: sender intent (E), sender's agent (As),
recipient's agent (Ar), recipient (R). This study asks where the irreducible core sits when everyone has
an agent. Do the two agents {As, Ar} become the core and displace both humans, or does a human stay in?

## What the lab already knows that bears on this

- **An open agent chain's core is a symmetric end pair (Q66, probes 229-230).** E → As → Ar → R is such a
  chain; its core should be a two-element end pair (a human and the adjacent agent), not the two agents.
- **A bidirectionally-coupled triage agent joins the core and displaces the sender (Q68, probe 232).** The
  recipient-side agent can take a human's place in the core. This study asks whether both agents can take
  both humans' places.
- **Frozen sources and sinks stay out; bidirectional coupling is necessary for core membership
  (Findings).** A human reduced to a frozen intent source or a passive readout should leave the core.
- **A closed loop binds the whole structure at higher Φ (Q66).** Closing the four-element exchange should
  bind all four.

## The gap

The lab has the one-sided agent (chain, triage) results but has not built the full two-sided exchange and
asked whether the two agents form the irreducible core with both humans displaced. No prior probe answers
the openclaw-to-openclaw question, so it is open.

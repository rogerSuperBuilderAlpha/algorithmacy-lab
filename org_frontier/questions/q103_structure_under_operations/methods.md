# Q103 — Stage 4 methods (fixed before computation)

Exact IIT-4.0 cause-effect structure via PyPhi's `phi_structure`, read at the integrating state. Run on
`~/iit-playground/venv-4.0/bin/python`.

## Forms (`forms.py`)

The read-recipient triad and three operations on it: substitution (Q64 substitutable at k=2), a tracking
memory (Q92), and delegation (the Q68 triage bidirectional form). For each, `joint_parties` returns the
system structure Φ, the dual type, and the set of parties appearing in any joint (two-or-more-party)
distinction of the integrated structure; a factored form (Φ = 0) carries no joint mechanism.

## Decision rules (`probe_structure_under_operations.py`)

- H1 confirmed if the substitutable form has Φ = 0 and no joint mechanism.
- H2 confirmed if the tracking-memory form has Φ > 0 and the memory node is in a joint mechanism.
- H3 confirmed if the delegation form has Φ > 0 and the triage agent is in a joint mechanism.
- H4 confirmed if, across all four forms, a joint mechanism is present exactly when Φ > 0.

# Q103 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on how the cause-effect structure changes under the program's operations. Each operation
has a known effect on the verdict: substitution collapses the triad (Q70), a tracking memory preserves it
while shifting the core to {M, R, Mem} (Q92), and recipient-side delegation puts the triage agent in the
core (Q68). This reads whether the integrated structure follows. Written and committed before the operation
forms are read. The baseline read-recipient triad (Φ = 2, mediated, joint mechanism {E, R}) is the
instrument control.

## H1 — Substitution collapses the integrated structure

- **Claim:** The substitutable form has system Φ = 0 and no joint mechanism binding the whole.
- **H0:** It retains a joint mechanism.
- **Predicted outcome:** collapsed. H0 refuted. An operation that collapses the verdict collapses the
  integrated structure.

## H2 — A tracking memory restructures the integrated structure

- **Claim:** The tracking-memory form keeps system Φ > 0, and the memory node appears in a joint mechanism.
- **H0:** Φ = 0, or the memory is absent from the joint mechanism.
- **Predicted outcome:** restructured, with the memory in. H0 refuted. The structure follows Q92's shift of
  the core to {M, R, Mem}.

## H3 — Delegation restructures the integrated structure

- **Claim:** The triage delegation form keeps system Φ > 0, and the triage agent appears in a joint
  mechanism.
- **H0:** Φ = 0, or the triage agent is absent.
- **Predicted outcome:** restructured, with the triage agent in. H0 refuted.

## H4 — Structure and verdict move together

- **Claim:** Across the operations, a joint mechanism is present if and only if the form is triadic
  (system Φ > 0).
- **H0:** Some operation leaves a joint mechanism without Φ, or Φ without a joint mechanism.
- **Predicted outcome:** they move together. H0 refuted. The integrated structure tracks the verdict at the
  collapse boundary, even though Q100 found its richness orthogonal to the scalar's magnitude.

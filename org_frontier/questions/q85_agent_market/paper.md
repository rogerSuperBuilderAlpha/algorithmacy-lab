# A market of interchangeable agents is a broadcast: interchangeability collapses outreach at every size

## Abstract

Q70 found that two interchangeable agents collapse the outreach triad. This study scales to a market of N
agents with exact IIT-4.0 Φ. Four pre-registered hypotheses are tested. A substitutable market, where the
worker and counterpart coordinate through any one of N agents, is dyadic at N = 2, 3, 4. An all-required
set is triadic at every N with Φ growing as 2N (4, 6, 8), and its major complex contains every agent. A
single substitutable pair among otherwise-required agents collapses the market to dyadic. All four
hypotheses confirmed: interchangeability collapses the coordination at every market size, and only a
jointly-required set of specific agents is irreducible.

## Introduction

Workers increasingly route outreach through whichever of many AI agents is available. Q70 tested two
interchangeable agents; this study scales to a market and asks whether interchangeability collapses the
coordination at every size.

## Related work

Platform research treats multi-homing across interchangeable providers as weakening any single one
[malone1994interdisciplinary]; the economics of spam frames cheap interchangeable sending as the dyadic
default [rao2012economics]; costly signaling frames commitment to a specific channel as informative
[spence1973job; zahavi1975mate]. Finding 5 holds that substitutability collapses the triad; integrated
information reads the verdict [oizumi2014phenomenology; albantakis2023iit4; mayner2018pyphi].

## Hypotheses

Fixed before computation (`hypotheses.md`). H1: a substitutable market is dyadic at every size. H2: an
all-required set is triadic with Φ growing. H3: the all-required core contains every agent. H4: one
substitutable pair collapses the market.

## Methods

Worker E, agents M1..MN, counterpart C; each agent reads E and C. substitutable: E and C read M1∨…∨MN;
all_required: E and C read M1∧…∧MN; mixed (N=3): one substitutable pair among required agents. Classified by
exact IIT-4.0 Φ; the all_required major complex is read for H3. Decision rules are in `methods.md`.

## Results

substitutable is dyadic (Φ=0) at N = 2, 3, 4. all_required is triadic at Φ = 4, 6, 8 for N = 2, 3, 4, with
the major complex {E, M1, M2, M3, C} at N = 3. The mixed form with one substitutable pair is dyadic. H1
through H4 all confirmed.

## Discussion

A market of interchangeable agents is structurally a broadcast. However many agents the market holds, if
the worker and counterpart can route through any one of them, no single agent is bound into the joint
determination and the coordination factors. Only a jointly-required set of specific agents is irreducible,
and then the integration grows with the set and every agent is in the core. A single swappable pair among
required agents is enough to collapse the whole market. The number of agents is not the variable; whether
each is required is. For agent-mediated outreach this completes the substitutability picture on the agent
side: a competitive agent market demands only literacy, and algorithmacy is demanded only when a specific
set of agents is jointly bound into the coordination.

## Limitations

The study is confirmatory. Results are in-silico: Boolean models, evidence about the models
[axtell1996aligning]. Φ is read ordinally and the 2N growth is a structural signature. Agents are
symmetric; heterogeneous agents are untested. Hypotheses were fixed before computing [chamberlin1965method;
platt1964strong; brodeur2024preregistration].

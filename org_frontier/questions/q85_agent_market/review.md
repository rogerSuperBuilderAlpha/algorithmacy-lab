# Q85 — Stage 1 review: a market of interchangeable agents

## The question

As AI agents proliferate, a worker can route outreach through whichever of many interchangeable agents is
available. Q70 showed that two interchangeable agents collapse the coordination. This study scales to a
market of N agents and asks whether interchangeability collapses outreach at every market size, while an
all-required set stays triadic. Parties: worker E, agents M1..MN, counterpart C.

## What the lab already knows that bears on this

- **Two interchangeable agents collapse the triad (Q70, probe 234).** Multi-homing across two agents is
  dyadic; requiring both stays triadic. This study scales N.
- **Substitutability of any role collapses the form (Finding 5); breadth dilutes random forms (Finding 6).**
  An interchangeable market is the substitutable case at scale.
- **An all-binding campaign stays triadic with Φ growing as n−1 (Q64).** The all-required agent set is the
  analogue on the agent side.

## The gap

Q70 tested two agents; the market at N = 2, 3, 4 is uncomputed, as is whether a single substitutable pair
among required agents collapses the whole. No prior probe scales the agent market, so the question is open.

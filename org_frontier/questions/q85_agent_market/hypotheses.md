# Q85 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on a market of N interchangeable worker-side agents (worker E, agents M1..MN, counterpart
C). Written and committed before any test runs.

## H1 — A substitutable market is dyadic at every size

- **Claim:** When E and C coordinate through any one of N agents (OR over the agents), the form is dyadic
  for N = 2, 3, 4.
- **H0:** Some substitutable market is triadic.
- **Predicted outcome:** dyadic at every N. H0 refuted.

## H2 — An all-required set stays triadic, with Φ growing

- **Claim:** When all N agents are required (AND over the agents), the form is triadic for N = 2, 3, 4,
  with Φ increasing in N.
- **H0:** Some all-required set is dyadic, or Φ does not grow with N.
- **Predicted outcome:** triadic at every N, Φ increasing. H0 refuted.

## H3 — The all-required core contains every agent

- **Claim:** The major complex of the all-required set contains E, C, and all N agents.
- **H0:** Some agent is outside the major complex.
- **Predicted outcome:** core is the full party set. H0 refuted.

## H4 — One substitutable pair collapses the market

- **Claim:** An otherwise all-required market with one substitutable pair among the agents is dyadic.
- **H0:** The mixed market is triadic.
- **Predicted outcome:** dyadic. H0 refuted.

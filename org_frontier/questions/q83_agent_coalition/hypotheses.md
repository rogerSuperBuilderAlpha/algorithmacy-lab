# Q83 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on a recipient-side gating coalition over the outreach triad (E, M, R), with two gating
agents T1, T2 that read the message and gate delivery to R. Written and committed before any test runs.

## H1 — Both required agents enter the core

- **Claim:** When delivery requires both agents (R'=M∧T1∧T2), both T1 and T2 are in the major complex,
  mirroring the regulator coalition.
- **H0:** Fewer than both agents are in the major complex.
- **Predicted outcome:** core contains T1 and T2. H0 refuted.

## H2 — A substitutable coalition keeps the agents out

- **Claim:** When either agent suffices (R'=M∧(T1∨T2)), the major complex does not contain both agents;
  the substitutable agents stay out.
- **H0:** Both substitutable agents are in the core.
- **Predicted outcome:** not both agents in the core. H0 refuted.

## H3 — A single bidirectional gating agent enters

- **Claim:** A single bidirectional gating agent is in the major complex, replicating Q68.
- **H0:** The single agent is outside the core.
- **Predicted outcome:** the agent is in the core. H0 refuted.

## H4 — The triadic core is present in every configuration

- **Claim:** The major complex is triadic (Φ > 0) for both-required, either-suffices, and single-agent
  forms.
- **H0:** Some configuration has a dyadic major complex.
- **Predicted outcome:** triadic core in all. H0 refuted.

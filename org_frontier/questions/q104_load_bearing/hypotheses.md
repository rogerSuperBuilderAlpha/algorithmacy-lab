# Q104 — Stage 3 hypotheses (fixed before computation)

Four hypotheses on the load-bearing element of the cause-effect structure — the cause-effect analogue of
Q93's structural margin. Q93 found the verdict fragile to a single bit-flip; this reads which distinction
carries the integration and which edge, removed, collapses it. Written and committed before the official
run; the instrument was validated on the read-recipient triad's distinctions and edge knockouts. Predictions
follow from Q99 (the integration lives in a binding mechanism) and Q93 (the triad is one edit from
collapse).

## H1 — The load-bearing distinction is binding, carrying φ_d = Φ

- **Claim:** The maximal-φ_d distinction is a binding one — its mechanism or purview spans two or more
  parties — and its φ_d equals the system Φ.
- **H0:** The maximal distinction is a single-party-to-single-party one, or its φ_d differs from Φ.
- **Predicted outcome:** binding, φ_d = Φ. H0 refuted.

## H2 — The singleton distinctions carry less than Φ

- **Claim:** Every distinction whose mechanism and purview are both a single party has φ_d below the system
  Φ.
- **H0:** Some singleton distinction carries φ_d equal to Φ.
- **Predicted outcome:** all singletons below Φ. H0 refuted. The parties individually carry less than the
  binding mechanism.

## H3 — Knocking the mediator's read collapses the verdict

- **Claim:** Removing the edge that enables the binding distinction — the mediator's read of a party —
  collapses the verdict to dyadic.
- **H0:** The verdict survives.
- **Predicted outcome:** dyadic. H0 refuted.

## H4 — Every edge is load-bearing

- **Claim:** Removing any single edge of the minimal triad collapses the verdict to dyadic — the triad has
  no redundancy.
- **H0:** Some edge can be removed with the verdict surviving.
- **Predicted outcome:** every edge load-bearing. H0 refuted, extending Q93's one-edit-from-collapse result
  to the read-recipient triad: not one edit but every edit collapses it.

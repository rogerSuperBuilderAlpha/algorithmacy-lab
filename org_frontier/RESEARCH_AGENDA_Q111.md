# Agenda: the political economy of the core (Q111–Q116)

A new angle. The program has read coordination structurally throughout — which parties are bound, what
shape the structure takes, how to build it. It has never asked the strategic question: who has power, who
captures value, who can extract rent. The structural results carry organizational stakes the program has
left implicit. Q97 found a coalition can capture the mediator, Finding 8 that a principal joins the core
only when bidirectionally coupled, Q98 that influence requires membership — all about power, none
quantifying it.

This wave bridges the structural account to cooperative game theory. The value of a coalition of parties is
the integrated information their subsystem sustains; a party's Shapley value distributes that worth by
marginal contribution. With value defined, the wave asks who captures it, who can veto the coordination,
how substitutability erodes power, and whether the principal's ownership buys a share. The questions use the
existing instrument on the forms already on the record, with the Shapley value computed exactly at these
sizes.

The wave gives the dissertation's organizational argument its missing half: not only when a coordination
demands the competency, but what being bound into it is worth.

## Q111 — The Shapley value of integration

**Question.** How is a coordination's value distributed among its parties? Does the mediator capture a
disproportionate share, and does a non-core party capture any?

**What it reads.** The Shapley value of each party, with the coalition value the subsystem's Φ.
*Discriminator:* the mediator's share against the egalitarian one, and a spectator's value against zero.

## Q112 — Veto power

**Question.** Which parties can unilaterally collapse the coordination? Q104 found every edge of the minimal
triad load-bearing; does that mean every core party holds a veto?

**What it reads.** The verdict after each party defects (sets its read to break the triad). *Discriminator:*
the set of parties whose defection collapses the form — the veto-holders.

## Q113 — Substitutability erodes power

**Question.** A substitutable party can be replaced. Does substitutability drop its Shapley value toward
zero, while a required party holds value?

**What it reads.** The Shapley values in a market of substitutable against all-required agents (Q85).
*Discriminator:* whether value tracks requirement, so an interchangeable party captures little and a
required one much.

## Q114 — The principal's value capture

**Question.** A principal who owns the mediator joins the core only when bidirectionally coupled (Finding 8).
Does joining the core let it capture value, and how much does it take from the parties?

**What it reads.** The Shapley redistribution when a principal enters the core. *Discriminator:* whether the
principal's value comes at the parties' expense (rent) or adds to the total.

## Q115 — The market's value distribution

**Question.** In a market of agents, how is value split between interchangeable agents and a required set?

**What it reads.** The Shapley values across the Q85 market configurations. *Discriminator:* whether the
required agents share value while the interchangeable ones capture little, the market reading of Q113.

## Q116 — Strategic value against structural contribution

**Question.** Does a party's Shapley value match its structural contribution — the φ_d it carries in the
cause-effect structure (Q104)?

**What it reads.** The Shapley value against the φ_d of each party's distinction. *Discriminator:* whether
strategic value and structural contribution are the same quantity, closing the loop between the
political-economy wave and the cause-effect-structure wave.

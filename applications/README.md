# Maintainer applications

This directory holds applications to join the project's review board. The role, the standard, and the
roster are in [`../MAINTAINERS.md`](../MAINTAINERS.md).

To apply: fork the repository, branch off `maintainer-applications`, copy
[`TEMPLATE.md`](TEMPLATE.md) to `<your-github-handle>.md`, fill it in, and open a pull request into the
`maintainer-applications` branch. The current maintainers review it in the open; a merge accepts it.

1. **Reproduction** — run the script, confirm the numbers, paste what you ran and what you got.
2. **Pre-commitment audit** — does the git history show hypotheses fixed before results, and are nulls

3. **Argument** — does the model represent what it claims, are the modeling commitments declared, does the

4. **Prose** — house style and whether citations resolve.
